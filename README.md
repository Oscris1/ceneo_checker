**Ceneo-checker** - Aplikacja pisana hobbistycznie napisana we Flasku. Wysyła maila do użytkownika gdy cena przedmiotu na ceneo spadnie poniżej podanej przez użytkownika wartości. Aplikacja posiada system logowania, każdy użytkownik posiada własną listę śledzonych przedmiotów. Użytkownik może dodawać przedmioty, zmieniać cenę poniżej której chce dostawać powiadomienia oraz usuwać przedmioty z listy. W przyszłości planuję zmienić scrappera z bf4 na scrappy oraz dodać śledzenie przedmiotów z innych stron.
